New Authentication Features Release Notes
NEW ADMIN GUEST PASSKEY

What Is This?

We have added a complete authentication and access control system to the VAC24 scoreboard. Members can now log in securely with passwords or passkeys, guest users can view the board read-only, and admin users have a dedicated management panel.

Key benefit: Before this change, anyone could edit or delete entries. Now only authenticated members can update their own progress, and only admins can manage system settings.

Login Methods

1. Password Login

  • Visit the scoreboard URL and you will see a login overlay.
  • Enter your member name (e.g., vacadmin) and password. The default password for all members is changeme.
  • After logging in, you can change your password from the user chip in the header (click the lock icon next to your name).
  • Your username is saved in browser localStorage, so refreshing the page restores your name in the login form.

2. Passkey Login (Passwordless)

  • Click Sign in with Passkey (green button on the login form).
  • You do not need to enter your name first. The browser will prompt you to select your passkey credential.
  • Authenticate using your device biometrics (Touch ID, Face ID, Windows Hello, PIN, etc.).
  • The system identifies you automatically and logs you in.

3. Guest Login (Read-Only)

  • Click Guest (Read-Only) on the login form.
  • No password required. You will have view-only access to the dashboard and member detail pages.
  • Guest users cannot submit updates, edit, delete, or access admin features.

Passkey Setup

Passkeys provide a secure, password-free way to sign in using your device biometrics or PIN. Each member can register multiple passkeys across different devices (laptop, phone, tablet).

How to Register a Passkey

  • Log in with your password first.
  • After logging in, click Passkey in the header actions.
  • On the passkey management page, click Register New Passkey.
  • Your browser will prompt you to create a credential. Follow the prompts (biometrics or PIN).
  • Your passkey is now registered. You can add more passkeys for additional devices.
  • To remove a passkey, click the delete button next to it in the list.
Important: Passkeys are tied to your device. If you lose access to a device, log in with your password and delete the old passkey from the Passkey page.

Admin Account

The admin account is vacadmin. Default password is changeme.

The vacadmin account and the guest account are filtered out from all statistics, charts, and team data. They do not appear in dashboards, leaderboards, or fun facts.

New Features Summary

Password Authentication

Secure bcrypt-hashed passwords. Default password is changeme for all members. Members can reset their password after logging in.

WebAuthn Passkeys

Passwordless login with device biometrics. Works on Mac (Touch ID), Windows (Hello), iOS (Face ID), and Android (fingerprint).

Guest Mode

No password required. Read-only access to view dashboards and member details. Cannot submit, edit, or delete.

Role Management

Admins can change member roles (member/admin) from the Admin panel. Only admins see the Admin button.

Session Cookies

httponly session cookies with 168-hour TTL. Sessions survive browser restarts.

Local Storage

Last login name is saved in browser localStorage. Pre-fills the username field on page load.

Deferred Submit on Bookmark

Bookmark member detail pages (e.g., /#detail/shek). After login, the update form opens automatically for that member.

Multiple Passkeys

Each member can register multiple passkeys across different devices. Existing passkeys are excluded from new registration to prevent duplicates.

Admin Stats Filter

vacadmin and guest accounts are excluded from all charts, leaderboards, and team statistics.

Quick Start

For Members

  • Your username is your member name (e.g., shek, nic).
  • Your default password is changeme.
  • Log in, then click the lock icon next to your name to change your password.
  • Set up a passkey on your primary device for faster login next time.

For Admin (vacadmin)

  • Log in with vacadmin / changeme.
  • Click Admin in the header to open the management panel.
  • Manage member roles, reset passwords if needed, and audit system data.
  • Change the default password immediately after first login.

Technical Details

Authentication Flow

Step Password Passkey
1. Begin POST /api/auth/login {name, password} POST /api/auth/passkey/auth/begin {name?}
2. Challenge Server validates password with bcrypt Server returns WebAuthn options (challenge)
3. Client Client receives {ok, name, role} Client calls navigator.credentials.get()
4. Finish Session cookie set POST /api/auth/passkey/auth/finish {credential}
5. Result Logged in Server verifies signature, identifies member, sets cookie

Database Changes

  • members table: Added role (member/admin/guest) and password_hash columns
  • passkeys table: Stores credential_id, public_key, sign_count, transports
  • audit_log table: Tracks data changes (prepared for future audit trail)
