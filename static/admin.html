Bulk Import

Upload a CSV to replace all updates for a specific member. Existing records for that member will be deleted first.

CSV Format Requirements
  • Required columns: date,name,module,description,status
  • Optional columns: leave_type (only for leave status), remarks
  • date: YYYY-MM-DD format
  • name: Must match the selected member exactly
  • status: Acceptable values: in_progress, done, leave, blocked, vague
  • All rows for other members will be rejected
date,name,module,description,status,leave_type,remarks 2026-05-13,ain,san,Implemented new feature,done,,Enhance Nebula TA to verify issue raised by pilot banks 2026-05-12,ain,myd,Team standup meeting,in_progress,, 2026-05-11,ain,dnp,Annual Leave,leave,AL,

Warning: This will permanently delete all existing updates for the selected member before importing the new records.
